# ITSystem - NODEJS

Полная копия сайта https://nodealit.netlify.app/, переделанная в React проект с использованием Vite.

## Технологии

- **React 19** - UI библиотека
- **TypeScript** - типизация
- **Vite** - сборщик и dev-сервер
- **React Router DOM** - маршрутизация
- **CSS Modules** - стилизация компонентов

## Структура проекта

```
src/
├── components/          # React компоненты
│   ├── Header.tsx       # Шапка сайта
│   ├── MainBanner.tsx  # Главный баннер
│   ├── About.tsx       # Секция "Обо мне"
│   ├── Advantages.tsx  # Преимущества
│   ├── Plan.tsx        # Программа курса
│   ├── Price.tsx       # Цены
│   ├── Footer.tsx      # Подвал
│   └── ...
├── assets/             # Статические ресурсы
│   └── 9ccdf6affc7ec8bbe371.png
├── App.tsx             # Главный компонент
├── main.tsx            # Точка входа
└── index.css           # Глобальные стили
```

## Установка и запуск

### Установка зависимостей
```bash
npm install
```

### Запуск в режиме разработки
```bash
npm run dev
```

Приложение будет доступно по адресу http://localhost:5173

### Сборка для продакшена
```bash
npm run build
```

### Превью продакшен сборки
```bash
npm run preview
```

## Компоненты

- **Header** - Навигационное меню с логотипом
- **MainBanner** - Главный баннер с описанием курса
- **About** - Информация об авторе
- **Advantages** - Преимущества курса
- **TextSection** - Текстовые секции
- **Plan** - Программа курса с аккордеоном
- **Price** - Секция с ценами
- **Footer** - Подвал с контактами

## Особенности

- Полностью адаптивный дизайн
- Плавная прокрутка к якорным ссылкам
- Интерактивный аккордеон для программы курса
- Современный UI с градиентами и анимациями
- TypeScript для типобезопасности
